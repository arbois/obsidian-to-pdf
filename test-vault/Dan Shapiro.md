# Dan Shapiro

Creator of [[Trycycle Overview|Trycycle]]. Built the hill-climbing dark factory approach on top of [[Superpowers|superpowers]] by [[Jesse Vincent]].

## Contributions

- Designed the plan-editing convergence loop (see [[Planning Deep Dive]])
- Implemented the ephemeral reviewer pattern (see [[Review Loop Mechanics]])
- Created the Python orchestrator for phase management
- Bundled EB Garamond font support for Typst output[^future]

> [!quote] On Hill Climbing
> "The trick is that each round is a fresh perspective. You're not asking someone to review their own work — you're asking a stranger to review a stranger's work. That's where the magic happens."
